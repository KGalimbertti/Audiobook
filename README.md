Audiobooks App (Web)

Este é um projeto simples de aplicativo web de audiolivros criado com HTML, CSS e JavaScript puros. Ele simula a interface básica de um player de áudio, permitindo visualizar a capa do livro, o título do capítulo e o autor, e fornece controles básicos de reprodução (tocar/pausar, avançar e retroceder).

Funcionalidades

Visualização de Informações: Exibe a capa do livro, o capítulo atual e o autor.

Player de Áudio: Toca um arquivo de áudio (.mp3).

Controles de Navegação:

Tocar/Pausar (botão central).

Avançar para o próximo capítulo (botão Próximo).

Retroceder para o capítulo anterior (botão Anterior).

Design Responsivo: (Assumido) Estrutura base para um design de player otimizado para dispositivos móveis (<meta name="viewport" ... />).****
